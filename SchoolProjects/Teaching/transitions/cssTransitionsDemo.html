Using CSS Transitions without Custom Libraries

Syntax

1. element { transition: [property] [duration] [timing-function: optional] [delay: optional] }

div { transition: background-color 2s linear 1.5s; }

2. element:hover { property: [value] }

OR

2. element:focus { property: [value] }

OR

2. element:active { property: [value] }

div:hover {background-color: blue;}
