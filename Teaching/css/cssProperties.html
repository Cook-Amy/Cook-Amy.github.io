Using classList Property

Here 2 of the 8 div sections before any changes

        <div class="changeDiv" id="changeFont" onclick="changeFont()">
              <p class="textDiv" id="fontText">Change the Font Type</p>
        </div>
        <div class="changeDiv" id="changeColor" onclick="changeColor()">
              <p class="textDiv" id="colorText">Change the Font Color</p>
        </div>

Here is the style section

        <style>
          .changeDiv {
            text-align: center;
            margin-left: 100px;
            margin-bottom: 10px;
            padding: 10px;
            width: 200px;
            height: 200px;
            background-color: yellow;
            border-radius: 0px;

            border: 0px; 
          }
          
          .allText {
            text-decoration: underline;
          }

classList.add("className") will add a predefined CSS class to an HTML element

      <script>
      function addStyling() {
        var textElement = document.getElementsByClassName("textDiv");
        for(var i = 0; i < textElement.length; i++) {
          textElement[i].classList.add("allText");
        }
      }
      </script>

      
Now the div sections have another class

      <div class="changeDiv allText" id="changeFont" onclick="changeFont()">
            <p class="textDiv" id="fontText">Change the Font Type</p>
      </div>
      <div class="changeDiv allText" id="changeColor" onclick="changeColor()">
            <p class="textDiv" id="colorText">Change the Font Color</p>
      </div>
    

    
    

      
classList.remove("className") will remove a CSS class to an HTML element

      function removeStyling() {
        var resetText = document.getElementsByClassName("textDiv");
        for(var i = 0; i < resetText.length; i++) {
          resetText[i].classList.remove("allText");
        }
      }

      
Now the div sections have only the original class

      <div class="changeDiv" id="changeFont" onclick="changeFont()">
            <p class="textDiv" id="fontText">Change the Font Type</p>
      </div>
      <div class="changeDiv" id="changeColor" onclick="changeColor()">
            <p class="textDiv" id="colorText">Change the Font Color</p>
      </div>
    


    

      
classList.toggle("className") will check if a predefined CSS class is present on an
 
        HTML element and then add or remove it

      function toggleStyling() {
        var toggleText = document.getElementsByClassName("textDiv");
        for(var i = 0; i < toggleText.length; i++) {
          toggleText[i].classList.toggle("allText");
        }
      }
      </script>
